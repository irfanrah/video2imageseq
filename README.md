
### Video to sequence image converter

1. Put your video in '/video_input'
2. In 'V2Iseq.py' you can tweak some parameter 'TOTAL_SEQ_IMAGE for number of image in sequence' and 'cv2.resize value'
3. Run 'python3 V2Iseq.py'
